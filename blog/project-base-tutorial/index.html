Project Base Tutorial

12 พฤษภาคม 2019

วันนี้ถ้าพูดในมุมของการพัฒนาระบบบนรูปแบบของ web application ต้องยอมรับว่า เทคโนโลยีต่างๆ หมุนเร็วมากๆ จนหลายๆครั้ง เราตามกันไม่ทัน และต้องยอมปล่อยหลายๆเรื่องราวให้หลุดมือไป ไม่ได้ไปจับ ไม่ได้ไปสนใจกับมัน เพราะเวลาที่มี ไม่อาจยัดทุกอย่างลงไปในหัวได้หมด แม้จะอยากทำสักเพียงไหน

วันหนึ่งถ้าเราเริ่มต้นเรียนรู้ เรื่องบางเรื่องใน version 2.5 พอเราเข้าใจดี จะเริ่มใช้งานแล้ว มันจะขยับเปลี่ยนเป็น version 3.0 ซะอย่างงั้น

แถมทุกครั้ง ผมจะพูดเรื่องพื้นฐานเสมอ เพราะถ้าพื้นฐานแน่นๆ เราจะขยับไปทางไหน มันไม่หนีไปจากพื้นฐาน รากแก่นแท้ ดั้งเดิมของมันมากนักอยู่แล้ว

แม้ตัวผมเอง เวลาจะเริ่มต้นอะไรใหม่ๆ คำว่า fundamental หรือเรื่องพื้นฐานดั้งเดิมต่างๆ จึงมักถูกหยิบมาเป็นเรื่องแรกๆที่ต้องเข้าไปคลุกกับมัน คอร์สเรียนต่างๆจะถูกหยิบขึ้นมาเริ่มต้นก่อนเสมอๆ

แต่จะบอกว่า แค่นี้มันไม่พอ ทันทีที่คุณบอกกับตัวเองว่า พื้นฐานกูแน่นพอแล้ว อยากเดินไปต่อแล้ว สิ่งที่คุณควรทำคือ การลงมือเริ่มต้นทำโปรเจ็คเลยครับ

ให้เริ่มต้นแบบง่ายๆก่อนเข้าไว้ จะเป็นระบบเล็กๆ หรือระบบย่อยๆจากระบบอื่นๆ อะไรก็ได้ มันจะทำให้คุณได้ใช้ หลากหลายกฏ หลากหลายความรู้ที่คุณได้เรียนมา ในคอร์สพื้นฐานต่างๆของคุณ

เหมือนเป็นการทบทวน ว่าที่เราเรียนมา เราเอาไปประยุกต์ใช้ได้จริงรึเปล่า รู้จักไหมว่าตรงไหนควรทำอะไร ทำอย่างไร และรู้วิธีแก้ปัญหา แก้บั๊ค การจัดการกับ Error ต่างๆทำอย่างไร

ระบบที่ผมมักหยิบมาเริ่มต้นก็จะเป็นระบบง่ายๆอย่าง ระบบจองห้องประชุม, ระบบผู้มาติดต่อ, ระบบตะกร้าขายสินค้า, ระบบเบิกค่าใช้จ่าย, ระบบ CMS, ระบบหวยนี่ก็เข้าท่า 555 คือแค่ลองทำนะครับ อน่าเอาไปใช้จริง

ระบบที่ทำหลายๆอย่าง ทั้งจัดการ session, cookie, มีหน้า report สำหรับค้นหา, มีระบบแบ่งแยกงานสมาชิกชัดเจน แม้จะมีเพียงไม่กี่หน้า บางระบบมีแค่ 2-3 หน้าก็อาจจะทำงานได้ครอบคลุมกับที่เรียนพื้นฐานมาแล้วด้วยซ้ำ

ตอนนี้ถ้าใครเข้าไปเรียน คอร์สพื้นฐานต่างๆมาแล้ว แต่ไม่รู้จะไปต่อยังไง ไม่แน่ใจว่าตัวเองเข้าใจมันลึกซึ้งดีพอรึยัง นาทีนี้บอกได้เลยครับว่า หยุดเรียนเลยครับ แล้วหันมาขึ้นโปรเจค ลงมือทำได้แล้ว

แล้วพอคุณทำมันเสร็จ คุณจะได้คำตอบทันทีเลยว่า คุณเข้าใจมันดีพอรึยัง ถ้ายัง ยังมีตรงไหนที่คุณควรหาข้อมูลอีก ยังมีตรงไหนที่คุณควรเสริม และเริ่มทำแบบยากขึ้นเรื่อยๆไปด้วยนะครับ ไม่เกิน 3 โปรเจค คล่องพอทำงานจริงแน่นอน
